Cities names breaks first

When the screen size is reduced, you'll notice that the city names are the first to wrap to a new line. Bootstrap's responsive design adjusts the column sizes based on the screen width. When size of the browser reduces, "col-md" class columns will stack vertically as these are medium sized screens, while the "col-sm-2" columns will maintain their horizontal arrangement for a longer duration.
